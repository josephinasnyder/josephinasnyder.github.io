Review of State Websites

State Vermont (Favorite)

  • This website was the best when talking about it's aesthetic
  • The search bar on the top and center with a beautiful picture of Vermont's fall following beneath it is inviting to the viewer.
  • The maneuverability is fantastic in comparison to the other websites
  • The load time was quick when shifting between the different links and options listed throughout the menu.
  • The website fits my computer screen fairly well.

    • This website is definitely the best when compared to the other four. I especially love how they did not put too much information on the home page like other websites. It is very easy to navigate through and the picture are very much inviting. I've been to the East Coast but never Vermont, maybe I need to look into it during my next trip!

    State Tennesse

  • The colors are a bit dark. However, I believe that this works for the website. It almost makes it appear more professional
  • This website was quick to load however Vermont's and Alaska's were a bit faster.
  • The links were easy to spot due to good pictures and bold lables. This made it easy to navigate between pages
  • The website stretches with my computer screen and phone. Very satisfying

    • This website probably was not a lot of peoples favorites but I liked how professional it looked. The main page was not too crowded and it was not difficult to look through the different pages when searching for something specific. It could improve, however, on it's load time which was either average of below average at times.

    State Iowa
  • The home page is incredibly busy and a little unsatisfying.
  • The website, for the most part, had a more modern appeal to it. However, it definitely lacks certain qualities.
  • The home page is set up in more of a listed format which makes the page long.
  • The load time was pretty average.
  • I did enjoy the quick links listed at the top of the page. This made it slightly easier to naviagate.
  • When clicking and looking through the other links, I was surprised to see that other pages (such as their "travel page") were more aesthetically pleasing than the home page.

    • Overall, this website leaves a lot to be desired. However, due to it's more modernistic appearance as well as it's aesthetically pleasing additional pages, I decided that it would fit in the 3rd spot following Tennesse. It also had an average load time. The home page should be further improved upon. Perhaps different colors would help to revert it from looking dull to looking interesting!

    State Alaska

  • There's a lot of controlled chaos on this website but still navigable
  • The color scheme could be furthur improved upon however it's pleasing how it follows the State's colors
  • The speed of the website was good. It was pretty quick when asked to load new pages
  • I liked the drop down menus that popped up when the curser sped over the top. If not easy, than manageable to navigate

    • There's definitely a certain aesethic about this website. However, it is not my favorite. I would rank it a 4. There's a lot I did not know about Alask but this website was informative and consumer friendly upon investigation. However, it could be further improved upon if updated to look more modern.

    State Oklahoma (Least Favorite)

  • The aesthetic of this website is definitely the worst out of all 5 websites.
  • The color are dull and not inviting. The first picture to greet the viewer is a boring field of grass which does not help.
  • The speed of the website was actually pretty good and actually surprising
  • There's hardly any information on the home page. The information that is present is hard to locate and find.
  • The drop down menus are formatted fairly well. However, it would help if the font was bigger.

    • This websites lacks any excitement and is not entirely inviting. The information that is given is difficult to find. However, I do find it interesting how the website really pushes for social media exposure. Perhaps those social media sights are formatted better than the initial website.
